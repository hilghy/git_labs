# Основные команды Git — кратко и с примером

## git init
Инициализирует новый локальный репозиторий.
Пример:
git init

## git clone
Клонирует удалённый репозиторий.
Пример:
git clone https://github.com/hilghy/git_labs.git

## git add
Добавляет изменения в индекс.
Пример:
git add README.md
git add .

## git commit
Создаёт коммит.
Пример:
git commit -m "Add README file"

## git push
Отправляет изменения в удалённый репозиторий.
Пример:
git push -u origin git-commands

## git pull
Загружает изменения с удалённого репозитория.
Пример:
git pull origin main
